
## Project Description

This FTOptix sample project demonstrates how to **Create Users from a JSON file stored on a USB** device. It provides an example of importing user accounts and credentials from an external JSON configuration file on removable media, enabling bulk user provisioning for HMI security management.

A .json example is in /ProjectFiles.
